## Personal Mood Dashboard App

## Features

/A fun and interactive web app that lets users express their mood, all with smooth animations and a motivational touch.\
\_Personal Greeting: — Enter your name and get a warm animated greeting.
\_Mood Box Control: — Grow or shrink the mood bubble interactively.
\_Toggle Day/Night Mode: — Switch between light and dark themes to fit your preference.
\_Secret Motivation Message: — Hidden message to keep you inspired and fired up.
\_Live Greeting Preview: — See your name appear as you type it.
